上記少年に対する頭書保護事件について、下記のとおり、付添人の令和○年○月○日付抗告申立書記載の抗告理由を補充する。

記

第１　少年はＡＤＨＤであること

# １　抗告申立書で述べたとおり、少年は発達障害の一種である注意欠如・多動性障害（ＡＤＨＤ）を有している。このことは、以下の事実から明らかである。

# ２　まず、少年は小学４年生で精神科を受診し、ＡＤＨＤとの診断を受けている。少年がＡＤＨＤの診断を受けたことは、社会記録中の児童相談所の回答書、少年の前件の鑑別結果通知書や少年調査票などからも明らかであり、これに反する証拠はない。

# ３　また、少年が小学４年生時に一時期通院していたことのあるカウンセリングルームで少年を担当していた心理カウンセラーも、通院時の主訴は、些細なことで瞬間的に怒り、粗暴な言動になること、授業に集中できず、立ち歩きなどが繰り返されていること、というものであり、こうした一連の問題行動はＡＤＨＤを持つ子の特性を表している、としており（意見書１頁）、カウンセラーも少年がＡＤＨＤであると診断している。

# ４　さらに、社会記録中でも、前述した児童相談所の回答書では、本児童の問題行動はＡＤＨＤの影響による衝動統制の未熟さである、とされている。

# ５　また、少年がＡＤＨＤであることは、少年の特性からも明らかである。

ＡＤＨＤの特徴として、①多動性、②不注意、③衝動性、の３つがあり、知的能力の割には学習効率がよくない、不器用、といったことがあるといわれている。

前述したカウンセリングルーム受診時の少年の主訴は、このようなＡＤＨＤの特徴と完全に合致している。

また、前件の社会記録中の小学校の学校照会においても、集中できなくなると、周囲の友だちに話しかけたりして、いたずらをしてしまうとあり、行動傾向の項で、自分の都合で衝動的に行動してしまい、相手の立場を考えられなくなることがあるとあり、さらに、中学校の学校照会でも、同様の行動傾向があるとされており、いずれもＡＤＨＤの特徴と完全に合致している。

このように、少年はＡＤＨＤの典型的な症状を有しており、この症状は後述するとおり、成長とともに変化は見られるとしても、基本的な部分は継続して持ち続けているのである。

# ６　本件の鑑別結果通知書及び少年調査票は、現在、障害は認められない、とか、医療措置不要（ＡＤＨＤと診断されたことはあるが、現在、障害は認められず、既往症と判定される）などと述べ、現在、少年がＡＤＨＤの症状を有していることについて否定的である。

しかし、これは、上記の医学的・心理学的診断があり、これと反対の医学的・心理学的診断が存在しないにもかかわらずＡＤＨＤではないと判断するものであり、何らの専門的根拠もなく、およそ信用できるものではない。

# ７　以上のとおり、少年がＡＤＨＤであることは疑いがなく、このことに触れていない原決定は要保護性を基礎づける重要な事実を看過している。

第２　ＡＤＨＤが本件非行に影響を及ぼしていること

# １　少年が有するＡＤＨＤは、本件非行（さらに言えば前件非行にも）にも大きな影響を及ぼしている。

# ２　前述したとおり、ＡＤＨＤは、①多動性、②不注意、③衝動性、の３つの特徴があるが、これらの特徴のうち①多動性は９歳前後でほぼ消失すると言われている。しかし、②不注意と③衝動性は中学生以降も残ることが多いとされており、また、その特徴のあらわれ方は子どもによって異なる。

# ３　少年の両親は幼少時より育てにくさを訴えており、前述したとおり、少年も小学校時代から教師や生徒とトラブルを起こしており、また、前件の調査官意見でも衝動性が強かったと評価されるなど、少年の場合も、不注意や衝動性及びそのコントロールの悪さが目立つ「多動性－衝動性優勢型」ともいうべきＡＤＨＤであるといえる。

本件も、少年が他のクラスの友人と話そうと思い、そのクラスの家庭科の授業中に家庭科室に入ったところを、そのクラスの担任教師にとがめられたことから、腹を立てて暴行に及んだものである。他のクラスの授業中に、授業をしている部屋に入るという一見非常識な行動も、その場で思いついたら止まらず、後先を考えずに突飛な行動をとるという衝動性のあらわれであり、少年のＡＤＨＤの特徴があらわれている。

そして、これをとがめられて教師に対して暴行に及んだのも、教師のＡＤＨＤに対する理解不足からくる規制的な対応により生じたものである。

# ４　これに対し、鑑別結果通知書や少年調査票は、過去にＡＤＨＤと診断されているが、現段階では、多動性や衝動性よりも感情統制の問題のほうが大きい、感情統制の悪さが改善されていないと指摘し、これを受けた原決定も、調査票を引用して、感情統制の悪さや暴力的行為を正当化する自己弁護的・他罰的な構えが強いという少年の傾向について、ＡＤＨＤの影響とはいえないと判示し、少年の感情統制の悪さや粗暴性に対するＡＤＨＤの影響を否定する。

たしかに、少年の暴力は感情統制の悪さが原因となっているものであり、この点は調査官意見や原決定が指摘するとおりである。しかしながら、感情統制の悪さは、少年の衝動性及びそのコントロールの悪さと、それを理解せずに規制的対応をとろうとする大人の対応のまずさに起因するものであり、少年は、ＡＤＨＤの衝動性を理解し、それを前提にして本人を落ち着かせてよく話を聞き、自分の行動を考えさせるというＡＤＨＤ児への基本的な対応をとる大人に対しては決して暴力的ではない。このことはカウンセラーが、たしかに怒りやすい傾向にはあるが、肯定的に接している大人とは大変素直に関係を持てている、としていることからも明らかである。

また、調査官意見や原決定は、少年が自分の考えで行動していることを理由に、感情統制の悪さ等がＡＤＨＤの影響ではないとする。

しかし、少年には統合失調症などの精神障害は認められないのであるから、少年が自分の考えで行動していることと、衝動性や感情統制が悪いこととは矛盾しない。いかなる行動も少年の考えに基づくものであるが、その「考え」がその場での思いつきだったり、自己の行動の結果をよく考えないままであったりするということであり（衝動性）、「考え」を持つことと怒りをコントロールできないこと（感情統制の悪さ）とは、まったく別次元のことである。調査官意見や原決定は、少年の衝動性や感情統制の悪さに対するＡＤＨＤの影響の理解を誤り、ひいては本件非行に対するＡＤＨＤの影響に関する事実及び評価を誤っているというしかない。

# ５　以上のとおり、少年のこれまでの暴力的な非行は、いずれもＡＤＨＤの特徴である衝動性や、それを無理に抑制することから生じる感情統制の悪さが大きな影響を及ぼしている。この点でＡＤＨＤの影響を否定する原決定は、要保護性を基礎づける重要な事実についての誤認があることは明らかである。

第３　少年の更生及び再非行防止にとって少年院送致はふさわしくないこと

# １　以上述べてきたとおり、ＡＤＨＤが本件非行に影響を及ぼしていることは明らかである。少年に必要なのはＡＤＨＤを克服して通常の社会生活が送れるようにするための精神医学的・心理学的治療・カウンセリングや、まわりの大人による、ＡＤＨＤの特性を踏まえ、少年の話をよく聞き、自分の行動を考えさせるという受容的な対応である。

# ２　本件を含むこれまでの少年の非行が、すべてＡＤＨＤの影響にあったことは前述したとおりであり、そのような問題解決の手段として、集団生活の下で規律や生活習慣を身につけさせる少年院送致の保護処分がまったく有効でないことは、本件が前件での少年院仮退院後まもなく発生したことからも明らかである。前述した少年の問題点である衝動性や感情統制の悪さは、原決定が判示するように、第一種少年院に収容し、所要の矯正教育を施すことなどでは解決できないのである。

# ３　少年の通っていた中学校は、早急に専門的な医療機関での入院治療を受けさせ、軽度発達障害の治療の両方ができる医療機関での入院治療が最適と考えていると述べ、前述したとおり、児童相談所も、本児童の問題行動はＡＤＨＤの影響による衝動統制の未熟さと考えられるため、本児童のＡＤＨＤ的傾向を理解した関わりを持ちながら、感情コントロールの訓練を行い、医療的ケアのある枠組みの中で処遇することが必要であると述べるなど、少年に関係した機関は一様に医学的ケアが必要であると判断しているのである。

# ４　また、本件の調査官意見も、少年院送致ではなく在宅試験観察であった。調査官は、①少年の粗暴行為が校内に限定されており、少年が家庭及び学校を中心とする保護領域から外れておらず、少年に個別的な指導への配慮を求める構えがあること、②中学卒業まで３カ月であり、高校進学を希望している少年にとっては自重につながること、③保護者が治療やカウンセリングを受けさせることを考えており、少年もこれを受け入れていること、④対人関係、とくに指導的立場にいる大人との関係改善ができるか否かがポイントとなるが、家庭裁判所の権威を少年、保護者とも認めているので、遵守事項を定め、学校等と情報交換しながら相応の改善を促すことが可能であること、を挙げて、在宅試験観察として在宅での更生のチャンスを与えることを考えたいと述べている。

このように、調査官も少年院送致ではなく、在宅処遇で少年が更生する可能性を認めていたのであり、かかる調査官意見はＡＤＨＤの影響について否定的な点を除けば少年の処遇として相当である。なお、上記③の点について、調査官意見は診療やカウンセリングの有効性は未知数であるとしているが、カウンセラーが述べるところによれば以下のとおりである。少年にとって必要なのは、自分に向き合い、自己肯定感を獲得し、信頼できる大人と出会い、他者との信頼関係を育てるということである。長期の集団による矯正教育ではなく、教育、医療と認知行動療法に基づく心理療法的治療が必要な支援であり、本人ならびに家族が望む方法でもある。治療教育を受け入れることが、少年が社会人としてよりよい成長を成し遂げていく出発点である。カウンセラーはこのように述べており、少年にとって精神医学的治療や心理カウンセリング・療法の効果があることは専門的に見て明らかである。

# ５　以上のとおり、再非行防止のための少年の処遇としては、ＡＤＨＤの治療と社会適応を図ることであり、それは精神医学的治療（投薬治療も含む）や心理カウンセリング・療法を行いつつ、まわりの大人がＡＤＨＤの特性を踏まえ、少年の話をよく聞き、自分の行動を考えさせるという受容的な対応を根気強くとっていくしかないのである。

具体的な処遇としては、保護観察、あるいはそれに不安が残るというのであれば、医学的・心理学的治療を受けることを遵守事項として在宅試験観察に付して様子を見るということが考えられる。なお、従前、少年の両親は少年に投薬治療を施すことに消極的であったが、現在は投薬治療を含むあらゆる医学的・心理学的治療を施すつもりでいる（両親の誓約書）。

しかるに、原決定は、以上の点を考慮せず、あるいは不当に軽視し、在宅試験観察で更生可能とした調査官意見も無視して、在宅処遇のままで行うことは著しく困難であると判示し、少年院送致とした。かかる原決定の判断は、前述した要保護性を基礎づける事実の誤認に起因し、保護観察あるいは在宅試験観察で十分に更生可能であり、かつ、それらが少年の更生にとって適切な処分であるにもかかわらず、かえって少年の問題を悪化させる危険すらある少年院送致とした点で、家庭裁判所の合理的な裁量の範囲を著しく逸脱したものである。

第４　原決定は憲法、子どもの権利条約、障害者差別解消法に反すること

# １　カウンセラーの意見書が指摘するとおり、本件非行は、保護者あるいは教師が付き添うなどの条件をつければ、問題なく参加できた少年の体育祭への参加を、そのような配慮を怠って許可しなかったこと、さらにそれに先行する一連の学校のＡＤＨＤ児である少年への配慮を欠いた対応が少年に不信と失望を増大させていったという、不適切な対応に起因する。そして体育祭への参加については、保護者あるいは教師が付き添うことがあらかじめ保護者の側から提起されていた。したがって、体育祭への参加に限ってみても、学校側にちょっとした心遣いがあれば、適切な対応は容易に実現できたものであった。原審の決定は、そのような合理的配慮を欠いたことに起因して、教師個人に向けられた少年の行為を非行として問題にしたものである。

しかし、この学校の対応は、明らかに合理的配慮を欠いた行為であり、以下に述べるとおり、我が国が１９９４年批准した児童の権利に関する条約（以下、「子どもの権利条約」という）が、差別として禁止している行為である。原審がそのような違法な禁止行為に起因する少年の行為を、その問題性への特段の配慮もないまま非行として、安易に再度第一種少年院に送致する処分をすることは、そのこと自体合理的配慮を欠いた行為として差別にあたり、許されない違法行為というべきである。再度の少年院への送致が、子どもの権利条約３７条（ｂ）の「拘禁」にあたることはいうまでもない。そして３７条（ｂ）は、拘禁について、「法律に従って行うものとし、最後の解決手段として最も短い適当な期間のみ用いること」としているのである。本件において再度の少年院送致が「最後の解決手段」でないことは、それが学校側の合理的配慮を欠いた行為に対する失望を回復させる方法として適切とはいえないこと、及び担当した家庭裁判所調査官が「試験観察」という違った処分意見を提出していることからしても明らかである。

# ２　障害を有する子どもには、合理的配慮を伴う対応が求められている。子どもの権利条約２３条２項は「特別の養護についての権利」を定め、同条３項は、その対応として「障害を有する児童が可能な限り社会への統合及び個人の発達（文化的及び精神的な発達を含む。）を達成することに資する方法で当該児童が教育、訓練、保健サービス、リハビリテーション・サービス、雇用のための準備及びレクリエーションの機会を実質的に利用し及び享受することができるように行われる」として配慮を求めている。そして、同条約２条１項は、「その管轄の下にある児童に対し、......心身障害、......にかかわらず、いかなる差別もなしにこの条約に定める権利を尊重し、及び確保する」（１項）とし、同条２項は、「あらゆる形態の差別又は処罰から保護されることを確保するためのすべての適当な措置をとる」として、差別を禁止しているのである。

この障害者についての差別に合理的配慮を欠くことが含まれることは、上記の子どもの権利条約２３条３項の文理だけではなく、２００６年１２月に採択され、２００７年９月２８日我が国が署名し、２００８年５月３日に発効した障害者の権利に関する条約（以下、「障害者権利条約」という）を受けて制定された、障害を理由とする差別の解消の推進に関する法律（障害者差別解消法）７条２項で、合理的配慮の提供が義務づけられていることからも明らかである。

# ３　こうした障害を有する少年の特性に対する合理的な配慮を否定する差別は、そもそも「様々な障壁との相互作用により他の者との平等を基礎として社会に完全かつ効果的に参加することを妨げ得るものを有する」者（障害者権利条約１条）への、人間としての尊厳の否定であり、日本国憲法１３条の「個人として尊重される」に違反する。また子どもの権利条約の否定は、日本国憲法９８条２項にも違反するものというべきである。

第５　結論

以上のとおり、原決定は、少年がＡＤＨＤであり、これが本件非行に多大な影響を及ぼしていることを看過して少年院送致の保護処分としており、要保護性を基礎づける重要な事実の誤認がある。

そして、少年にはＡＤＨＤ克服のための医療的・心理学的な治療を施すことが最もふさわしいことは前述したとおりである。ＡＤＨＤの子どもの再非行防止の観点からは少年院送致はふさわしくなく、かえって問題を悪化させる危険すらある。これは裁判所の合理的な裁量の範囲を著しく逸脱したものであり、原決定の第一種少年院送致の保護処分が著しく不当であることは明白である。

よって、原決定は取り消されたうえ、直ちに○○家庭裁判所に差し戻されるべきである。
